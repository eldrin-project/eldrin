About Eldrin

What is Eldrin?

Eldrin is a modular backend system written in Rust, designed to be extensible through modules. It provides a robust foundation for building scalable and secure web applications.

Key Features

  • Modular Architecture: Easily extend functionality with custom modules.
  • Secure Authentication: Built-in support for email/password, Google, and GitHub authentication.
  • High Performance: Built with Rust for maximum performance and safety.
  • API-First Design: RESTful API endpoints for seamless integration with any frontend.
  • Database Integration: SQLx-powered database interactions for type-safe queries.
